ADBridge setup

1. Extract this zip before running ADBridge.
2. Run ADBridge.exe from this folder.
3. Do not move ADBridge.exe out of this folder by itself.
4. Keep the scrcpy folder beside ADBridge.exe for mirroring to work.
5. If antivirus quarantines scrcpy.exe, restore it into this extracted folder.
